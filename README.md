# Titanic Dataset Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand passenger characteristics, identify patterns, detect missing values, and determine factors affecting survival.

## Objectives
- Explore dataset structure
- Analyze missing values
- Identify trends and patterns
- Test hypotheses
- Detect anomalies and data quality issues

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Titanic Passenger Dataset

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers were more likely to survive.
- Higher fare passengers had better survival chances.
- Age showed a relatively weak relationship with survival.
- Significant missing values were found in Cabin, Body, Boat, and Home Destination columns.
