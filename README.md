# CodeAlpha_Titanic_EDA
Titanic Dataset Exploratory Data Analysis
Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, missing values, and factors affecting passenger survival.

Objectives
Understand dataset structure
Detect missing values
Identify patterns and trends
Test hypotheses
Discover factors influencing survival
Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Key Findings
Female passengers had significantly higher survival rates.
First-class passengers were more likely to survive.
Passengers paying higher fares had better survival chances.
Age showed a weaker relationship with survival.
Several columns contained missing values.
Dataset

Titanic Passenger Dataset
